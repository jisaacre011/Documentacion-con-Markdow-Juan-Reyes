Documentación con Markdown — Juan Reyes
Este repositorio contiene mi trabajo de documentación profesional usando Markdown, hecho para la materia de Administración de Base de Datos.

¿Qué hay aquí?
El archivo principal es documentacion_markdown_Juan_Reyes.md, donde investigué qué es Markdown, su sintaxis, lo comparé con otras herramientas y armé una guía práctica de comandos Git.
Contenido del documento

Parte 1 — Investigación: qué es Markdown, para qué sirve, ventajas, limitaciones y toda la sintaxis básica con ejemplos.
Parte 2 — Comparación: Markdown frente a Notion, Obsidian y Google Docs.
Parte 3 — Práctica: una guía de comandos Git para principiantes.
Parte 4 — Reflexión: por qué Markdown sigue siendo tan usado y si seguirá vigente.

Cómo lo organicé:
Escribí todo en un solo archivo .md para que fuera fácil de leer y exportar. Usé encabezados para separar cada parte, listas y tablas para ordenar la información, bloques de código para los ejemplos y una imagen para explicar el flujo de trabajo de Git. Lo edité en VS Code y lo subí a GitHub para tenerlo con control de versiones.
